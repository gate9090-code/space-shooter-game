===============================================
우선순위 (효과가 큰 순서)
===============================================

1순위: Shockwave (shockwave_ring.png)
   → 보스전에서 매우 눈에 띔

2순위: Particle (particle_spark.png, particle_smoke.png)
   → 가장 많이 사용되는 효과

3순위: Spawn Effect (portal_core.png, portal_ring.png)
   → 적 스폰 시 인상적인 효과

4순위: Bullet (bullet_glow.png)
   → 플레이어가 계속 보는 효과

5순위: Coin/Gem (coin.png, heal_gem.png)
   → 수집 동기 부여


===============================================
이미지 제작 가이드
===============================================

✅ 권장 포맷: PNG (투명 배경 필수)

✅ 크기: 자유롭게 제공 가능
   - 코드에서 자동으로 크기 조정
   - 고해상도 (512x512 이상) 권장
   - 축소는 쉽지만 확대는 화질 저하

✅ 색상:
   - 밝고 선명한 색상 권장
   - 애디티브 블렌딩 효과 고려

✅ 투명도:
   - 가장자리는 투명하게
   - 중심부는 불투명하게
   - 부드러운 그라데이션

✅ 여러 버전 제공 가능:
   - particle_spark_1.png
   - particle_spark_2.png
   - particle_spark_3.png
   → 코드에서 랜덤 선택 또는 용도별 사용


===============================================
참고사항
===============================================

• 모든 이미지는 선택 사항입니다
• 일부만 제공해도 괜찮습니다
• 기존 원형 방식이 폴백(fallback)으로 작동
• 이미지 로드 실패 시 자동으로 원형으로 전환
• 크기가 다르거나 비율이 달라도 자동 조정
