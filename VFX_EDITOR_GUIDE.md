# VFX Editor - 아티스트를 위한 사용 가이드

## 🎨 개요

**VFX Editor**는 프로그래밍 지식 없이 **드래그 앤 드롭 + 슬라이더**만으로 게임 효과를 만들 수 있는 툴입니다.

---

## 🚀 빠른 시작

### Step 1: 툴 실행
```bash
python vfx_editor.py
```

### Step 2: 이미지 준비
- PNG 파일 (투명 배경 권장)
- 권장 크기: 360x360px 이상

### Step 3: 효과 만들기
1. PNG 이미지를 **왼쪽 패널**로 드래그
2. **슬라이더**로 효과 조정
3. **오른쪽 프리뷰 영역 클릭**으로 테스트
4. 마음에 들면 **저장 버튼** 클릭

## ⚙️ 설정 항목

### 1. **카테고리** (드롭다운)
- `hit_effects` - 총알 적중 효과
- `critical_effects` - 크리티컬 효과
- `boss_effects` - 보스 전용 효과
- `skill_effects` - 스킬 효과

### 2. **크기** (50 ~ 600)
- 효과의 최대 크기 (픽셀)
- 작은 값 = 작은 효과
- 큰 값 = 큰 효과

### 3. **지속시간** (0.1 ~ 3.0초)
- 효과가 화면에 보이는 시간
- 짧은 값 = 빠르게 사라짐
- 긴 값 = 천천히 사라짐

### 4. **색상 R/G/B** (0 ~ 255)
- 이미지에 입히는 색상 틴트
- `255, 255, 255` = 원본 색상
- `255, 0, 0` = 빨간색
- `0, 255, 0` = 초록색
- `0, 0, 255` = 파란색

### 5. **파동 개수** (1 ~ 10)
- 연속으로 나타나는 효과 개수
- 1 = 단일 효과
- 3+ = 파동 효과

### 6. **파동 간격** (0.01 ~ 0.5초)
- 파동 사이 시간 간격
- 짧은 값 = 빠르게 연속
- 긴 값 = 천천히 연속

## 🎮 조작법

### 마우스
- **드래그 앤 드롭** - 이미지 로드
- **프리뷰 영역 클릭** - 효과 생성
- **슬라이더 드래그** - 설정 조정

### 키보드
| 키 | 기능 |
|----|------|
| `SPACE` | 자동 재생 ON/OFF |
| `C` | 모든 효과 제거 |
| `ENTER` | 설정 저장 |
| `ESC` | 종료 |

### 버튼
- **저장** - JSON 파일로 저장
- **효과 클리어** - 화면의 모든 효과 제거
- **자동 재생** - 자동으로 효과 계속 생성

---

## 💾 저장 방법

### 1. 설정 조정 완료
- 슬라이더로 원하는 효과 만들기

### 2. 저장 버튼 클릭 (또는 ENTER 키)
- 자동으로 `assets/config/vfx_effects.json` 업데이트

### 3. 개발자에게 알림
```
✓ 효과 저장 완료!
  카테고리: hit_effects
  이름: my_custom_effect

→ 게임에서 사용:
  vfx.create_multi_shockwave(pos, 'hit_effects', 'my_custom_effect')
```

이 정보를 개발자(당신)에게 전달하면 됩니다!

## 📋 작업 흐름 예시

### 시나리오: 불 속성 효과 만들기

1. **이미지 준비**
   - `fire_ring.png` 제작 (빨간/주황색 링)

2. **VFX Editor 실행**
   ```bash
   python vfx_editor.py
   ```

3. **이미지 로드**
   - `fire_ring.png`를 왼쪽 패널로 드래그

4. **설정 조정**
   - 카테고리: `hit_effects`
   - 크기: `280`
   - 지속시간: `1.0`
   - 색상 R: `255`
   - 색상 G: `100`
   - 색상 B: `50`
   - 파동 개수: `5`
   - 파동 간격: `0.08`

5. **테스트**
   - 프리뷰 영역 클릭
   - 자동 재생 ON으로 계속 확인

6. **저장**
   - ENTER 키 또는 저장 버튼

7. **개발자에게 알림**
   > "불 효과 완성했어요!
   > 카테고리: hit_effects
   > 이름: fire_ring"

## 🎨 팁과 요령

### 이미지 준비
1. **투명 배경** 사용 (PNG 필수)
2. **정사각형** 이미지 권장
3. **중심에 효과** 배치
4. **가장자리 여백** 10% 이상

### 색상 틴트
- `255, 255, 255` = 원본 유지
- 밝은 색 = 강렬한 효과
- 어두운 색 = 은은한 효과
- `0, 0, 0` = 검정색 (보이지 않음 주의!)

### 파동 설정
- **단발 효과**: 파동 1개, 간격 0
- **리플 효과**: 파동 3-5개, 간격 0.1초
- **연속 효과**: 파동 7-10개, 간격 0.05초

### 크기 가이드
| 용도 | 권장 크기 |
|------|----------|
| 일반 적중 | 200-300 |
| 크리티컬 | 300-400 |
| 보스 효과 | 400-600 |
| 스킬 효과 | 300-500 |

## 🐛 문제 해결

### Q: 이미지를 드래그했는데 안 보여요
A:
- PNG 파일인지 확인
- 파일 크기가 너무 크지 않은지 확인 (10MB 이하 권장)
- 다시 드래그해보세요

### Q: 효과가 안 보여요
A:
- 색상이 `0, 0, 0` (검정)이 아닌지 확인
- 크기가 너무 작지 않은지 확인 (최소 50 이상)
- 프리뷰 영역 안쪽을 클릭했는지 확인

### Q: 저장이 안 돼요
A:
- `assets/config/vfx_effects.json` 파일 권한 확인
- 이미지를 먼저 로드했는지 확인
- 콘솔 메시지 확인

### Q: 자동 재생이 너무 빨라요/느려요
A:
- 코드에서 `auto_spawn_interval` 값 조정 (기본 1.0초)
